# CS115 Quiz Grader
A Java console application that allows a student to enter up to 10 quiz scores using grades 60–100, computes the average and displays the letter grade based on the average.

## GitHub Repository
[https://github.com/rgwhite210/cs115-quiz-grader](https://github.com/rgwhite210/cs115-quiz-grader)

## How to Run
**Requirements:** 
Java JDK installed

**From the command line:**
```bash
git clone https://github.com/rgwhite210/cs115-quiz-grader.git
cd cs115-quiz-grader/src
javac QuizGrader.java
java QuizGrader
```
## Usage
- Enter quiz grades one at a time (valid range: 60–100)
- Type `999` to stop entering grades
- The program will display all entered grades, the average, and the letter grade
